# Targets audit (Sofia dogfood)

> **Not a scorecard.** A honesty check: your stated intentions add up to more than a week holds. That's normal—some are **aspirations**, some are **ceilings**, some **overlap**.

## Weekly targets (explicit)

| Area | Target | Type |
|------|--------|------|
| Social life (lavender) | **10h/week** | Floor (insight if below) |
| Self-care (lavender) | **10h/week** | Floor — showers, skin/hair |
| DJing & music prep | **10h/week** | Floor |
| Exercise (sage) | **5h/week** | Floor |
| Meals & nourishment | **~14h/week** | Band (not guilt) |
| RBL volunteer | **2× 2–3h blocks** (~4–6h) | Rhythm |
| Laundry | **2h/week** | Floor |
| Admin | **Under 4h/week** | **Ceiling** (insight if over) |
| Job search & brand | **3–4h/day** | **Trimester aspiration** — not a weekly budget |
| Home cleaning | **10h/month** | ~2.5h/week equivalent |
| Day rest | **Max 2h/day** | Ceiling (siesta/podcast) |

## Rough sum (if everything hit at once)

| Bucket | Hours |
|--------|-------|
| Social + self-care + DJing | 30h |
| Meals + exercise + RBL + laundry | ~25h |
| Job search *(if 3h/day × 7)* | **~21h** |
| **Subtotal** | **~76h+** |

A week has **168h**; sleep and work aren't on this list. So many **10h floors** together is ambitious—not wrong, but they will **trade off**.

## Contradictions & tensions (not failures)

1. **Job search “3–4h daily” vs everything else**  
   Taken literally (~21–28h/week), it dominates. In the product, treat this as **trimester direction**, not a weekly insight threshold. Weekly “thin week” stays at **&lt;2h scheduled**.

2. **Three 10h pillars** (social, self-care, DJing)  
   Coherent emotionally; mathematically tight alongside job search and work. Insights should **observe**, not imply you can max all three every week.

3. **Related blocks double-count in the chart**  
   CAPRIXXO → RBL + DJing; lavender errand → social + RBL. **Intentional** so nothing disappears—not double guilt.

4. **Self-care vs exercise**  
   Showers on **lavender → self-care**; gym/yoga on **sage → health**. Different areas; both body, different intent.

5. **Evenings open / free mornings**  
   Structural goals (space), not extra hours—compatible with everything.

## Recommendations (product, not life advice)

- Keep **ceilings** (admin, day rest) as ceilings.
- Keep **trimester** language for job search in goals, not daily failure cards.
- In UI later: show **“this week's focus”** (pick 2–3 priorities), not all floors at once.
